        ######
Client Side - React with Context API
Server Side - Node js (ExpressJs framework)
Database - MongoDb


API folder for server Side
clientApp for Client Side


##Install and Running Process:

#Backend Side:
    At first open the API (Server Side) folder in Visual Studio Code.
    
    then, writting the command 'npm install' in the terminal or command line.
    
    waiting for installing all needed dependencies, they all install and store in node_modules folder.
    
    After that, run the command in terminal or Command Line 'node index.js' .
    
#Frontend Side:
    open the clientApp in Visual Studio Code.
    
    run the command 'npm install'
    
    then run 'npm start' command.
    
Finall Client side communicate with Backend Server.
